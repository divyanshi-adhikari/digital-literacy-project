#  Task 2 – Digital Portfolio Description

I created my digital portfolio using three platforms: GitHub, LinkedIn, and Kaggle. 
GitHub is mainly used to showcase coding projects and maintain repositories, which will help me store and present my work. 
LinkedIn is a professional networking platform where I added my educational details, and it will help me connect with professionals and explore opportunities. 
Kaggle is a platform for data science and machine learning, where I can practice problems, participate in competitions, and improve my analytical skills.

Setting up these profiles helped me understand the importance of having a professional online presence.
